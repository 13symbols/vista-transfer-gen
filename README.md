# vista-transfer-gen
# 🏷️ Transfer Cards Generator (Logistics)

**Professional tool for generating transfer cards. Optimized for A4 printing with automatic 10-card layout.**

---

### 🌍 English Description
Automate your logistics documentation with this specialized generator. It creates standardized transfer cards with pre-defined locations.

* **Smart Layout:** Automatically fits 10 cards (2x5 grid) onto a single A4 sheet.
* **Location Presets:** Quick selection of departure and arrival points.
* **Vector Quality:** High-quality PDF export for crisp printing.
* **Fast Workflow:** Generate a full sheet of cards in seconds.

---

### 🇺🇦 Опис (Українська)
Професійний інструмент для генерації карток переміщень. Створено для автоматизації документообігу між складами та магазинами.

#### 🚀 Можливості:
* **Розумна верстка:** Автоматично розміщує 10 карток (сітка 2х5) на одному аркуші А4.
* **Локації:** Швидкий вибір пунктів відправлення та призначення зі списку.
* **Якість:** Генерація чіткого PDF-файлу, готового до друку.
* **Швидкість:** Підготовка повного аркуша карток за кілька кліків.

#### 🛠 Як користуватися:
1. Оберіть пункт **«Відправник»** та **«Отримувач»**.
2. Перевірте вигляд картки у вікні попереднього перегляду.
3. Натисніть **«Завантажити PDF»** або **«Відкрити PDF»** для друку.
